# Direkta — Small Smart Features Roadmap

## Photos & Media
- [ ] **Drag-and-drop to reorder photos** — seller wants best photo first for cover
- [ ] **Photo quality dots** — green/yellow/red indicator on each photo thumbnail based on qualityScore
- [ ] **Lightbox zoom** — click photo to see full-size in overlay
- [ ] **Bulk delete** — select multiple photos and delete at once
- [ ] **"Best cover photo" suggestion** — AI picks the best exterior shot automatically

## Property Page
- [ ] **Auto-save indicator** — small "Gespeichert ✓" toast after every change
- [ ] **Progress bar** — visual % of how complete the listing is (use completenessScore)
- [ ] **"Preview as buyer" button** — see how the listing looks on the public landing page
- [ ] **Copy/duplicate property** — create a similar one without re-entering everything
- [ ] **Days since created** — visible on property detail page header

## PDF & Sharing
- [ ] **Share via WhatsApp** — one-click share button generating a direct link
- [ ] **Email PDF** — send PDF directly from the app to a recipient
- [ ] **QR code on PDF** — auto-generated QR linking to the online listing page
- [ ] **"Nicht geprüft" warning** — banner before PDF download if AI content wasn't seller-reviewed

## Pricing & Market
- [ ] **Price change history** — graph/timeline showing when and how the price was changed
- [ ] **"Days on market" counter** — visible on listing card and dashboard
- [ ] **EUR/m² comparison** — "Your price: 4,200 EUR/m² | Market avg: 3,800 EUR/m²"
- [ ] **"Price too high" nudge** — warning if asking price is >15% above recommendation

## Leads & Communication
- [ ] **Quick-reply templates** — "Vielen Dank fuer Ihr Interesse..." one-click responses
- [ ] **"Away" mode** — auto-reply when seller is on vacation
- [ ] **Lead notification badge** — red dot on sidebar when new inquiry arrives

## General UX
- [ ] **Activity log** — "Photo uploaded", "Price changed", "PDF downloaded" timeline on property page
- [ ] **Undo last delete** — accidentally deleted a photo? Undo within 10 seconds
- [ ] **Mobile photo upload** — scan QR code on desktop, upload photos from phone camera
- [ ] **Keyboard shortcuts** — Ctrl+S to save, Ctrl+P for PDF download
